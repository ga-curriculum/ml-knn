## About
Understand what classification is and learn how to implement k-Nearest Neighbors (KNN) classification in Python using the `scikit-learn` library.

## Learning Objectives

*You will be able to:*

- Understand how classification works.
- Describe the k-NN algorithm and the intuition behind it.
- Prepare data for k-NN algorithm using standardization techniques.
- Fit, evaluate, and generate predictions from a k-NN model in `scikit-learn`.

- Have familiarity with `scikit-learn`.
